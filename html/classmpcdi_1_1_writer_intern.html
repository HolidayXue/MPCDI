Detailed Description

Internal Functions for the Writer.

Date:
2012-02-08 11:39:41 -0500 (Wed, 08 Feb 2012)
Revision:
18341

Part of the reason of having this internal class is that it makes sure that all the 3rd party libraries that don't need to be exposed in mpcdi, are hidden to the user.
